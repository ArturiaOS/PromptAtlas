# PromptAtlas

**PromptAtlas** is an open repository for sharing high-quality prompts for Large Language Models (LLMs).

The goal is simple:

```text
Build a public "atlas" of effective prompts that anyone can use, learn from, and contribute to.
```

## Why PromptAtlas?

Working with LLMs is often about how you ask, not just what you ask.

Good prompts can:

* Dramatically improve output quality
* Save time and iteration
* Unlock advanced use cases

But most great prompts are:

* Scattered across chats
* Lost in personal notes
* Hard to discover or reuse

PromptAtlas solves that by making prompts:

* Discoverable
* Reusable
* Comparable
* Shareable
